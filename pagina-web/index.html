Programación

Programación es la acción de ordenar, estructurar o componer una serie de acciones cronológicas para cumplir un objetivo. La programación puede ser aplicado en la creación de aplicaciones, videojuegos, bases de datos, software, etc.

En informática, la programación es el uso de lenguajes informáticos para imprimir en un sistema computacional procesos y funciones deseadas.
